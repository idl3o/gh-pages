Development Commands

Use these commands to develop and maintain the project:

npm run start or VS Code task "Start Jekyll (Ruby 2.7)" to serve the Jekyll site locally.
npm run build to build the Jekyll site into the _site directory.
npm run spellcheck to check for spelling errors.
npm run docs to generate API documentation using TypeDoc.
npm run verify-docs to check for missing documentation files.
