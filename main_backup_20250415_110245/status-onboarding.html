Welcome to the Status Page

This onboarding guide will help you understand how to use the Web3 Crypto Streaming Service status page effectively.

Understanding Status Indicators

The status page uses colored indicators to show the health of various system components:

Operational - System is functioning normally
Degraded Performance - System is experiencing issues but still working
Outage - System component is not functioning

Key Components

Monitor the health of these critical platform components:

  • Blockchain Connectivity - Connection to Ethereum and other chains
  • Smart Contract Status - Health of deployed contracts
  • Content Delivery - Video streaming infrastructure
  • API Services - API endpoint availability
  • Authentication - Wallet connection services

Incident History

The status page displays recent incidents and their resolution status. Click on any incident for details about:

  • Root cause analysis
  • Steps taken to resolve the issue
  • Preventative measures for the future

Setting Up Notifications

Stay informed about system status changes:

  1. Click the "Subscribe to Updates" button
  2. Choose your preferred notification method
  3. Enter your contact details
  4. Confirm subscription through the verification link
Pro Tip: You can customize which components you want to be notified about in the notification settings.
